git checkout -b avance (para crear y moverse a una nueva rama llamada avance)
git add . (guardar los cambios)
git commit -m "Avances en la funcionalidad x" (registrar los cambios)
git push -u origin avance (subir la rama avance)

Cuando los avances estén listos y revisados:
git checkout main (para cambiarme a la rama principal main)
git pull origin main (para actualizar con lo último)
git merge avance (une los avances desde la rama)
git push origin main (subir el resultado)
